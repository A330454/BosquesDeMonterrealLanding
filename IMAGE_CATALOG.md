## Performance Optimization Notes

### Image Optimization Strategies Used:
1. **Lazy Loading**: Applied to all `<img>` tags below the fold
2. **Responsive Sizing**: Images requested at appropriate widths for their display size
3. **Quality Compression**: All images use `q=80` quality parameter
4. **CSS Background Images**: Used for decorative elements to separate content from presentation
5. **Fixed Attachment**: Used sparingly on hero sections for parallax effect

### Recommendations for Future Optimization:

## Technical Implementation Details

### Image Formats:
- **JPEG**: All Unsplash images (via URL parameters)
- **SVG**: Favicon and inline icons
- **Embedded Content**: Google Maps iframes

### URL Parameters Used:
- `w=` : Width specification (600, 800, 1920)
- `q=` : Quality setting (80 for all images)

### CSS Properties Applied:
- `background-size: cover` - Ensures full coverage
- `background-position: center` - Centers images
- `background-attachment: fixed` - Parallax effect on heroes
